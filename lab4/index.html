Pseudocode Baking a Cookie

  • 1. Find a recipe
  • 2. Get ingredients
  • 3. Be in a kitchen
  • 4. Follow the instructions in the recipe
  • a. Measurements
  • b. Mixing
  • c. Temperature of the oven
  • 5. Eat the Cookies!
  • 6. VOILA!

Pseudocode a Game: Hangman

  • 1. Print hangman platform
  • 2. Generate a random word from a predefined list
  • 2.5 Print dashes for number of letters in word
  • 3. Accept user guesses
  • 4. If correct print letter in correct space
  • 4.5 If not correct print body parts in order: head, body, left arm, right arm, left leg, right leg, game over
  • 5. If game over, restart and generate new word
  • 6. If user guesses word within the six guesses, print "Congratulations!" and full body walking away
